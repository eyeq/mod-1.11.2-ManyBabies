# [子沢山]ManyBabies

[**ダウンロード**](https://github.com/eyeq/mod-1.11.2-ManyBabies/releases/download/1.0/1.11.2-ManyBabies-1.0.jar)

動物が繁殖するときに、複数生まれるようになります。  
繁殖数は、動物によって異なります。  
大きく増えるのは、ブタ、ウサギ、オオカミ、ヤマネコです。
